> 只要接触 Linux ，不可避免的会用到 Vim 编辑器。一方面它基本上是所有 Linux 发行版的默认编辑器；另一方面它是一个超级强大的文本编辑器。

```bash
vim +5 file #打开文件并跳到第5行，无指定数字则跳到最后一行
vim +/abc file #打开文件并搜索 `abc`，按 n 切换至下一个 `abc`
vim file1 file2 file3 #打开3个文件，按 `:n` 切换至下一个文件，`:N` or `:prev` 切换至上一个文件

  <----- a、i、o    ↓↓    : ----->
输入模式          命令模式          底行模式
          ESC ----->       <----- ESC

底行模式常用指令
:w     保存
:q     退出
:wq     保存并退出
:x     保存并退出
:!     强制执行
:ls     列出打开的所有文件
:n     切换至下一个文件
:N     切换至上一个文件
:15     跳到第15行

命令模式常用指令
h     j     k     l
左   下   上   右
HOME     移动到行首
END     移动到行尾
PgUp     向上翻页
PgDn     向下翻页
Ctrl+B     向上翻页 back
Ctrl+F     向下翻页 front
dd     删除(剪切)光标所在的一整行
5dd     删除(剪切)光标处开始的5行
yy     复制光标所在的一整行
5yy     复制光标处开始的5行
p     在光标所在行的下方粘贴
P     在光标所在行的上方粘贴
o     在光标所在行的下方插入一行并切换到输入模式
O     在光标所在行的上方插入一行并切换到输入模式
/abc     从光标位置向后搜索 `abc`
?abc     从光标位置向前搜索 `abc`
n     显示搜索命令定位到的下一个字符串
N     显示搜索命令定位到的上一个字符串
x     删除光标当前所在位置的字符
s     删除光标当前所在位置的字符并进入编辑模式
u     撤销上一步的操作
Ctrl+R     恢复上一步的操作
ZZ     保存并退出
gg     跳到第一行
G     跳到最后一行
H     跳到屏幕第一行
M     跳到屏幕中间一行
L     调到屏幕最后一行


Vim 的配置文件为 /etc/vimrc
set number     显示行号
set cursorline     高亮当前行

Tips：
yum -y install vim #安装 Vim
vim /root/.bashrc #设置别名，替换系统默认编辑器
      alias vi=vim
大部分 Linux 发行版默认编辑器是 Vi，一个简化版的 Vim。
```
