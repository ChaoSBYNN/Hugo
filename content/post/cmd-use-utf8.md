## 1. 修改 CMD 页面编码
```
chcp 65001 #改用 UTF-8 编码
```

## 2. 修改 CMD 页面字体
```
在标题栏上点击右键，选择"属性"->"字体"，将字体修改为True Type字体"Lucida Console"。
```

---

## 其他编码格式
```
chcp 437 #英文
chcp 932 #日文
chcp 949 #韩文
chcp 936 #简体中文
chcp 950 #繁体中文
```
